# Scientific Calculator

Basic & Scientific calculator implemented in Java using JavaFX.

Includes two themes:
* Orange & Grey
* Dark

Includes 2 angle modes:
* Degrees
* Radians

As well as:
* Hypotenuse mode
* FE mode
